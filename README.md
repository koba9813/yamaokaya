# 山岡家ナビ (Android)

現在地を取得し、現在地から最寄りの山岡家までの距離と方角を表示するアプリです。


## 実装内容
- 位置情報権限のリクエスト (Fine/Coarse/バックグラウンド)
- 現在地のリアルタイム取得 (FusedLocationProviderClient)
- 端末方位のリアルタイム取得 (回転ベクトルセンサー)
- 事前登録した山岡家の座標リストから最寄り店舗を検索
- 最寄り店舗を距離で判定
- 画面中央の矢印を店舗方向へリアルタイム回転
- 距離 (km) と方角 (16方位 + 角度) をリアルタイム表示
- 最寄り店舗までの距離を常時通知（Foreground Serviceによるバックグラウンド更新対応）
- 同一店舗へのチェックインは3時間に1回のみ可能
- 10m/50m以内で特別な演出（カオスなラーメン画像アニメーション）

## 構成
- Kotlin
- Jetpack Compose
- Google Play Services Location

## 実行方法
1. Android Studio でこのフォルダを開く
2. Gradle Sync を実行
3. 実機またはエミュレータで起動
4. 位置情報許可を与える

## 注意
- 検索対象はアプリ内に事前登録した店舗座標です。
- 検索半径は 100km に設定しています。
- バックグラウンドでの距離通知には「常に許可」の位置情報権限と、端末のバッテリー最適化除外設定が必要な場合があります。
- Foreground Serviceを利用しているため、通知が常時表示されます。
